# Customer Purchasing Preferences

This project explores factors that influence customer purchasing preferences using a retail shopping dataset containing 5,000 customer transactions.

The analysis combines Python-based exploratory data analysis (EDA), statistical hypothesis testing, and an interactive Power BI dashboard to identify patterns in customer behavior.

## Research Questions

- Does gender influence product category preferences?
- Does age influence purchasing preferences?
- Does season influence product choices?
- What products and categories are most popular among customers?

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- SciPy
- Power BI

## Key Findings

- Gender significantly influences product category preferences.
- Seasonal differences exist in specific product purchases.
- Footwear is the most popular product category overall.
- Age showed limited influence on purchasing preferences.
